# Color - RgbClash

![License](https://img.shields.io/badge/License-MIT-blue.svg)

This is a collection of smart contracts built for the Color - RgbClash application. These contracts enable the creation, management, and trading of unique Color and organizing RgbClash games on EVM based blockchains.

## Table of Contents

- [Features](#features)
- [Links](#links)
- [License](#license)

## Features

- **NFT Contract**: Manages the creation and ownership of unique Color NFTs.
- **NFT Pouch**: Allows users to stake their NFTs and earn rewards.
- **Game Contract**: Organizes games and handles the distribution of rewards to participants.
- **Security**: Utilizes OpenZeppelin contracts to ensure robust security and best practices.
- **Reentrancy Guard**: Protects against reentrancy attacks in critical functions.
